# Airbnb Exploratory Data Analysis (EDA)

This project performs Exploratory Data Analysis (EDA) on an Airbnb dataset.

## Overview
This repository contains a Jupyter Notebook that explores the Airbnb dataset to identify trends, missing data, and other insights. The analysis utilizes popular Python data science libraries such as Pandas, NumPy, Matplotlib, Seaborn, and Missingno.

## Setup & Requirements

To run the notebook, ensure you have the required libraries installed. You can install all dependencies using `pip`:

```bash
pip install numpy pandas matplotlib seaborn missingno
```

## Running the Notebook
1. Clone this repository.
2. Ensure you have the dataset downloaded (the notebook reads from an `airbnb.csv` file). You will likely need to adjust the file path in the notebook to point to your local `airbnb.csv` file.
3. Open `text_generation.ipynb` using your preferred Jupyter Notebook environment.
